Создание репозитория
git init

Добавить глобального пользователя, есть локальные в каждом репозитории
git config --global user.name "agallardo"

Добавляем глобальный email
git config --global user.email "agallardo@yandex.ru"

Кофигурационные данные
git config --list

Внесенные данные
C:\Users\agal\.gitconfig

Игнорирование папок и файлов прописывается в:
.gitignore

Текущий статус проекта
git status

Показать файлы, которые еще не в git
git status --untracked-files=all

Добавить все файлы папки в git
git add .

Удалить файл из под присмотра git
git rm --cached file.php

git commit -a -m"init"
-a все файлы из под присмотра
-m сообщение

Добавить под присмотр untracked file - проиндесировать
git add command.txt

Комиит файлов проиндексированных
git commit -m"add files"

Добавить в индекс все модифицированные php-файлы
git add "*.php"

Добавить в индекс отдельный файл
git add command.txt

Если файл модифицирован и не проиндексирован, то он в commit не уйдет

Откат к прошлой версии файла
git checkout -- file.php

Просмотр прошлых команд
git log

Выход из просмотра информации в консоли "Q"

Помощь - страница в интернете (по функции log)
git help log

Более интересный вид отображения информации в консоли
git log --pretty=format:"%h - %an, %ar : %s"
%h - короткий код коммита
%an - автор
%ar - время операции
%s - комментарий

Информация за определенное время
git log --since=2.week

Последний commit и изменения при проведении commit
git log -p -2
